# Alura Tracker
Projeto desenvolvido na trilha de Vue.

- Vue 3.0

Link do projeto hospedado na vercel: https://alura-tacker.vercel.app/

## Passos para executar o projeto
```
npm install

npm run serve
```
## Build do projeto

```
npm run build
```

## Lints e corrigir arquivos
```
npm run lint
```
